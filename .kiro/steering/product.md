# Product Overview

Progressive is a gym workout tracking mobile application built with Flutter. The app enables users to:

- Authenticate and manage their accounts securely
- Track workouts and exercises
- View workout history and progress
- Calculate plate weights for barbell exercises
- Manage user profiles

The app connects to a Spring Boot backend API for data persistence and user management. It uses JWT token-based authentication with secure local storage for session management.

Target platforms: Android and iOS mobile devices.
